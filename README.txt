A lightweight script that moves the currently active window between virtual desktops on Windows, using AutoHotkey and VirtualDesktopAccessor.​

Steps:
1. Download AutoHotkey.
2. Launch the script.

How to use:
Shift + Ctrl + Win + Left Arrow → Moves the active window to the desktop on the left.
Shift + Ctrl + Win + Right Arrow → Moves the active window to the desktop on the right.

Launch on pc startup:
1. win + r.
2. Write "shell:startup".
3. Move a shortcut of the "move-window2.ahk" script into that folder.
